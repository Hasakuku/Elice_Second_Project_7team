커밋 컨벤션
- [ ] `Feat` : 새로운 기능 추가
- [ ] `Fix` : 버그 수정
- [ ] `Docs` : 문서 수정
- [ ] `Style` : 코드 포맷팅, 세미콜론 누락, 코드 변경이 없는 경우
- [ ] `Refactor` : 코드 리펙터링
- [ ] `Test` : 테스트 코드, 리펙터링 테스트 코드 추가
- [ ] `Chore` : 빌드 업무 수정, 패키지 매니저 수정
- [ ] `Hotfix` : 간단한 코드수정
예시: Feat: "회원 가입 기능 구현"

코딩 컨벤션

DB
- [ ] Model 파일명은 대문자
- [ ] 필드명은 CamelCase 사용
- [ ] Boolean 타입은 is를 접두사로 사용
